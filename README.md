Running the GSM.swf Simulation

Download the file GSM.swf.
Open the following link: https://flashplayer.fullstacks.net/?kind=Flash_Emulator
Click Choose File and select GSM.swf.
Set the display parameters:
Width: 800
Height: 400

Specify the simulation parameters:
Number of particles: e.g., 100
Rotation speed of the disk: 40
Angle between the slots on the two disks: 20
Simulation duration (minutes): 6
Start the simulation and observe the results on screen.
